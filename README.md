## Features

## Technologies
Project is created with:
* React: 18.2.0
* Vite: 4.1.0
* Sass: 1.58.3

## Setup
To run this project, install it locally using npm:
```
$ cd ../memmory-game
$ npm install vite
$ npm run dev
```
![Local server IP](/images/readme_screenshot.png)

Paste the specified IP address into your browser
